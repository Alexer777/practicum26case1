Описание метода.
Шаги:
1.	Генерация сетки: Создание векторной сетки 1×1 км (для примера, на территории агломерации города Новосибирска).
2.	Майнинг признаков: Скачивание данных OSM для трех слоев (Наука, Индустрия, Демография) на основе тегов.
3.	Расчет мульти-индекса: Вычисляет весовые коэффициенты для каждой ячейки на основе площади объектов и их типа.

Логика расчёта весовых коэффициентов:

1.	Индекс Науки (W_sci) 
Научные объекты часто компактны (занимают несколько зданий), поэтому для повышения значимости необходимо присваивать ячейкам с объектами науки более высокий вес. Применяем логарифмическое преобразование, для того, чтобы даже малая площадь давала высокий балл.
2.	Индекс Индустрии (W_ind)
Используется линейная доля площади, т.к. промышленные показатели (выбросы, потребление энергии) обычно прямо пропорциональны площади территории завода.
3.	Индекс Демографии (W_demo) 
Зависит от площади жилой застройки. Используется линейная доля застройки.
